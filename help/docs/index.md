# 介绍

本项目用于数据仓库构建，数据处理，etl等，所用工具如下：

- **python**：数据处理与数据分析
- **azkaban**：任务调度管理
- **mysql等数据库**：数据仓库与数据存储
- **其他**：
    - **FineReport**：自动化报表构建

# 目录说明

    azkaban         # azkaban调度文件
    code            # python数据处理文件
    data            # 数据源
    help            # 帮助文档
    source          # 公共函数
