## 防火墙
```
# 配置防护墙
vi /etc/sysconfig/iptables

# 增加
-A INPUT -p tcp -m state --state NEW -m tcp --dport 3306 -j ACCEPT
-A INPUT -p tcp -m state --state NEW -m tcp --dport 8081 -j ACCEPT

# 启动防火墙
service iptables status

```
